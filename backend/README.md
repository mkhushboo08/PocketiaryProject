## Features
- CRUD operations on notes  

- **Resource Highlighter**  
  Important parts of a note, such as keywords, categories, or recently updated content, are visually highlighted. This helps users quickly identify and focus on relevant information without reading the entire note.

- **App Open Streak Tracking**  
  The application tracks consecutive days the user opens the app. This streak feature encourages regular journaling and helps build a habit of daily diary usage by motivating users through consistency.

- **Calendar Integration**  
  Notes are linked with a calendar view, allowing users to:
  - See which days have diary entries  
  - Navigate notes by date  
  - Quickly access past entries by selecting a specific day  

  This makes it easier to reflect on previous experiences and maintain a timeline of thoughts.

- **Side Panel Notes View**  
  All notes are displayed in a side panel for quick access. Users can:
  - Instantly switch between notes  
  - See note titles at a glance
 


  - Improve navigation without leaving the current view
